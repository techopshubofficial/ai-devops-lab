# AI DevOps Lab

Building AI tools for DevOps engineers — one episode at a time.

These are **not** ChatGPT wrappers. Each tool reads your real files and infra,
extracts the facts that matter, and gives context-aware answers about *your*
setup, with best practices and risks.

## Episodes

| Episode | Project | Command |
|---------|---------|---------|
| [EP01](./ep01-devops-ai-cli) | DevOps AI CLI | `ai explain deployment.yaml` |

More coming: Terraform review, Dockerfile generator, Kubernetes troubleshooter,
incident RCA, PR reviewer, and a full DevOps agent.
